# libpassman
Library for interacting with passman++ databases.

# Building
Simply run:
```bash
$ cmake -S . -B build
```

To install, run (as root):
```bash
# cmake --build build --target install
```

Also available from the AUR as `libpassman`:
```bash
$ git clone https://aur.archlinux.org/libpassman.git
$ cd libpassman
$ makepkg -si
```

Or with yay:
```bash
$ yay -S libpassman
```
